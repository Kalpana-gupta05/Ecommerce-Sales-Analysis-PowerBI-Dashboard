# Ecommerce-Sales-Analysis-PowerBI-Dashboard
End-to-End Power BI project on e-Commerce Sales dataset.

## Project Overview
This project is an end-to-end analysis dashboard built using Power BI.
It provide insights into ecommerce sales performance, customer behavior, and profit trends using interactive visualizations.

## Dashboard Highlights
-Total Sales Amount: 438k
- Total Profit: 37k
- Toal Quantity Sold: 5615
- Average Order Value(AOV): 121k

## Key Insights
- Maharashtra and Madhya Pradesh contribute the highest sales.
- Clothing category dominates with 63% of total quantity sold.
- COD (Cash on Delivery) is the most used payment mode(44%).
-  Monthly profit shows fluctuations with some loss months.
-   Printers and Bookcases generate the hightest profit among sub-categories.

## Visualization Included
Used various visualization types such as bar chart, pie chart, donut chart, line chart, and clustered chart.

- Sales by State
- Quantity by Category
- Profit by Month
- Sales by Customer Name
- Quantity by Payment Mode
- Profit by Sub-Category
